# Roadmap

Versioned, prioritized. Updated monthly.

## v1.0 — Foundation (Current)

- [x] 12 core skills across 3 tiers (Workflow, Quality, PM Bridge)
- [x] Official SKILL.md format compliance
- [x] One-line install script
- [ ] Demo recordings (plan-first flow, debug-circuit in action)
- [ ] Cross-platform testing: Codex CLI, Gemini CLI

## v1.1 — Orchestration

- [ ] Cross-skill examples: plan-first + ship-cycle + verify-build as a complete workflow
- [ ] Example project walkthroughs in `/examples`
- [ ] Skill composition guide: how to combine skills effectively

## v1.2 — Quality Infrastructure

- [ ] Automated skill linting (validates SKILL.md frontmatter and structure)
- [ ] Eval harness: test skills against sample prompts, measure behavior adherence
- [ ] Security audit of all skills (document findings publicly)

- [ ] `trade-estimator` — Construction bid generation skill (HVAC, electrical, plumbing)
- [ ] `supply-chain` — Supply chain analysis and optimization
- [ ] Domain-agnostic skill template for building custom domain packs

## v2.1 — Distribution

- [ ] `npx claude-skillcraft install` CLI
- [ ] Plugin marketplace submission
- [ ] Selective install: `install.sh --tier workflow` or `install.sh --skill plan-first`

## Future Considerations

- Enterprise pack: compliance, audit trail, approval workflows
